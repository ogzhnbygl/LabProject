# LabProject - Proje ve Etik Yönetim Sistemi

Laboratuvar projelerini, etik kurul izinlerini ve hayvan kotalarını yöneten merkezi modül.

## Özellikler
- Proje Takibi (Başlangıç/Bitiş Tarihleri)
- Etik Kurul İzinleri
- Hayvan Kotası Yönetimi (Dispo ile entegre)

## Kurulum
Dispo ile aynı altyapıyı kullanır.
```bash
npm install
npm run dev
```
